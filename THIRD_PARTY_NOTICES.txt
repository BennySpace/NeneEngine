NOTES
=====

- DiligentEngine contains its own third-party subcomponents. The primary
  project license is Apache 2.0, but additional notices for bundled
  subcomponents are present under the external/DiligentEngine tree. If you
  redistribute that subtree or binaries produced from it, review those nested
  license files as well.

- spdlog notes a dependency on fmt. In the current build, fmt is present as a
  transitive dependency in the vcpkg_installed tree.

- The bundled Momosuzu Nene model is sourced from Sketchfab and requires
  attribution under CC BY 4.0. The local asset license file also records the
  original source URL and author name.

- The vcpkg_installed paths above are build-tree paths for the current
  workspace. If the build directory is regenerated elsewhere, the referenced
  files may move, but the corresponding package copyright files should still
  exist in the matching vcpkg_installed/<triplet>/share/<package>/ directories.
